# Lobotomy Corporation Thai Translation Remastered
addon แปลไทยสำหรับ Lobotomy Corporation

เป็นการนำม็อด [**Thai Translation**](https://www.nexusmods.com/lobotomycorporation/mods/923?tab=description) มาแปลเนื้อหาใหม่ เพื่อให้สอดคล้องอารมณ์ของตัวละคร โดยตามอ้างอิงจาก [Library Of Ruina ภาษาไทย : Reprinted](https://steamcommunity.com/sharedfiles/filedetails/?id=3153033050)

อยู่ขั้นตอนในการแปลใหม่ทั้งหมดโดยใช้ Base ของม็อดไทย นำมาแปลใหม่ทั้งหมด

# เนื้อเรื่องที่ไม่สามารถแปลได้ 
ต้องทำความเข้าก่อนว่า เนื้อเรื่อง "เมล็ดพันธุ์แห่งแสงสว่าง 100% (Seed of light 100%)" ไม่สามรถแปลได้จริงๆ เนื่องจากว่าไฟล์ของเนื้อเรื่อง อยู่ในไฟล์เกม ไม่ใช่ไฟล์ xml ดังนั้นผมเลยไม่สามารถแปลได้จริงๆ ต้องขอโทษด้วยครับ
